Política de Privacidad - MiMismoV1

Última actualización: 23 de abril de 2025

1. Información que recopilamos

MiMismoV1 es una aplicación interna de SITWIFI, S.A. de C.V. que únicamente recopila la siguiente información:

  • Correo electrónico corporativo para fines de autenticación
  • Datos básicos del perfil para identificación del usuario

2. Cómo utilizamos la información

La información recopilada se utiliza exclusivamente para:

  • Autenticar usuarios con cuentas corporativas de SITWIFI (@sitwifi.com)
  • Proporcionar acceso a la información relevante para sus funciones laborales
  • Mantener un registro de sesiones por motivos de seguridad

3. Compartición de datos

No compartimos ninguna información personal con terceros. La aplicación es de uso interno para personal autorizado de SITWIFI.

4. Seguridad

Implementamos medidas de seguridad para proteger la información, incluyendo:

  • Tokens de autenticación con tiempo limitado de validez
  • Restricción de acceso sólo a dominios corporativos
  • Registro de intentos de inicio de sesión

5. Derechos del usuario

Los usuarios tienen derecho a:

  • Acceder a sus datos
  • Solicitar modificaciones de su información
  • Revocar permisos de acceso
